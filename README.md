Микросервис для администраторов закрытых Telegram-каналов. Полностью автоматизирует рутину: самостоятельно одобряет входящие заявки на вступление и мгновенно отправляет кастомное приветственное сообщение пользователю в ЛС.

## 🚀 Функционал
* **Мгновенный авто-прием:** Обрабатывает событие `ChatJoinRequestHandler` без задержек.
* **Приветствие в ЛС:** Бот автоматически пишет новому подписчику сразу после одобрения заявки.
* **Повышение конверсии:** Помогает удерживать аудиторию в приватных каналах за счет моментального отклика.

## 🛠 Стек / Технологии
* Python 3
* aiogram 3 (Asynchronous Telegram Bot Framework)
* asyncio

## ⚙️ Как использовать

1. **Создайте приватный канал** в Telegram и в настройках («Пригласительные ссылки») создайте ссылку с включенным пунктом **«Заявки на вступление»**.
2. **Добавьте бота в администраторы** вашего канала с правом *"Добавление подписчиков"*.
3. **Установите зависимости** из файла `requirements.txt`
   ```bash 
   pip install -r requirements.txt
