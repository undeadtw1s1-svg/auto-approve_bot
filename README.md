# ⚡️ Бот для авто-приема заявок в Telegram-канал

Микро-сервис для администраторов закрытых каналов. Автоматизирует рутину: самостоятельно одобряет заявки на вступление и мгновенно отправляет приветственное сообщение в личные сообщения пользователю.

## 🛠 Стек:
* **Python 3**
* **aiogram 3**

## 🚀 Как использовать:
1. Добавьте бота в администраторы вашего канала с правом "Добавление подписчиков".
2. Включите в канале "Заявки на вступление".
3. Запустите скрипт — бот начнет принимать заявки.
